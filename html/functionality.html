Summary of Functions

Cliniface's interface is made up of menus and toolbars along the top of the main window, the viewing region for the 3D images, and a dropdown list of the open files at the bottom of the window with buttons either side for copying and moving models between Cliniface's three viewers (only one viewer is visible initially). Any greyed out button corresponds to a currently unavailable action (usually because a model isn't selected). The buttons on the toolbars show only the most commonly used functions. The menus can be used to access nearly all available functions in Clinface.

Cliniface's functionality is described below grouped by parent menu category. Some functions are only available in the context of making and analysing measurements. More details on these functions can be found in the measurements section.

File

Open a 3D image.

Save the active model at its current file location.

Save As. Save the active model under a different file name/location. Save models in the 3DF file format to retain the metadata Cliniface needs for its analysis. See here for more information on the 3DF file format.

Import Metadata in XML or JSON format and overwrite the active model's metadata. See here for more information on the metadata import/export format.

Export Metadata from the currently selected model to CSV, JSON, or XML format. See here for more information on the metadata import/export format.

Undo the last action carried out.

Redo the last action carried out.

Assessment Information. Open the Assessment Information dialog where subject demographics and assessment notes can be entered. Demographic data about subjects is needed for Cliniface to know which measurements are atypical for flagging HPO terms.

Model Properties. Display a summary of the active model's geometry.

Create Template Report. Create and save a PDF report about the active model.

Close the active model.

Close All currently open models.

Quit. Close all models and quit.


View

Texture. Toggle visibility of the model's texture map (or return to surface view).

Wireframe. Toggle visibility of the geometric wireframe mesh.

Manifold Boundaries. Toggle visibility of surface manifold outlines.

Show Correspondence Mask. Toggle visibility of the correspondence mask (anthopometric mask). Only available after landmark detection; it shows the result of the non-rigid anthropometric mask registration process.

Median Plane. Toggle visibility of the mid-sagittal (medial) plane (after landmark placement).

Transverse Plane. Toggle visibility of the transverse plane (after landmark placement).

Select Radial Area. Interactively select a region around a point. Use in combination with the face extraction tool to remove unwanted parts of the model. Click and drag the centre handle to move the extraction region around and use forwards and backwards on the mouse wheel to change the radial size of the region.

Triangle Labels. Toggle visibility of the individual polygon identifiers.

Vertex Labels. Toggle visibility of the individual vertex identifiers.

Backface Culling. On by default, this prevents the inside facing surface of the model from being rendered. Sometimes, the inside and outside surfaces of a model are incorrectly defined due to the ordering of vertices for each triangle. In this case Cliniface's invert normals tool can be used to swap the inside and outside surfaces of the model.

World Axes. Toggle visibility of the world axes grid.

Take Screenshot (All Viewers). Save a screenshot of the viewers. If the left or right viewers are in use, the images from them are horizontally concatenated into a single image before saving.

Show 3D (Red/Blue). Toggle anaglyphic 3D rendering. Red/blue anaglyph glasses are needed to see the 3D effect.

Visualisations Toolbar. Toggle the visualisations toolbar on and off.


Surface Mapping (View submenu)

Scalar Legend. Toggle the legend on and off when visualising scalar mappings of the model's surface.

Asymmetry [All] (mm) Shows the relative difference in three dimensions between the two lateral halves of the face according to the anthropometrically mapped mask. Asymmetry is shown as positive or negative. If positive, that area is tumid relative to the corresponding area on the opposite lateral. If negative, it is shrunken relative to the opposite lateral. By default, asymmetry is shown in five colour bands of severity within the range +/- 5.0 mm (2 mm per band) but this can be changed from the visualisations toolbar.

Asymmetry [X] (mm) Shows lateral asymmetry through the mid-sagittal plane of the face only (superior/inferior and anterior/posterior asymmetry not shown).

Asymmetry [Y] (mm) Shows vertical (superior/inferior) asymmetry only (lateral and anterior/posterior asymmetry not shown).

Asymmetry [Z] (mm) Shows depth (anterior/posterior) asymmetry only (lateral and superior/inferior asymmetry not shown).

Mean Curvature (degs). Shows average curvature in degrees around each polygon (triangle). This is an average of the curvature at each triangle's three vertices inversely weighted by the proportional areas of adjacent triangles. Negative values are convex and positive values are concave. The sign of the curvature depends upon the orientation of the surface normals. The range of curvature displayed by default is +/- 45 degrees in 6 degree bands though these settings can be changed from the visualisatios toolbar.

Absolute Curvature (degs). Shows the absolute value of the mean curvature. By default this is shown in the range [0,45] degrees in 9 bands of 5 degrees each.

Right (+X) Orientation. Shows scalar values proportional to the magnitude of each triangle's surface normal projected along the positive X axis (the global X axis if no landmarks are present to provide alignment, otherwise the X axis given by the facial alignment).

Up (+Y) Orientation. Shows scalar values proportional to the magnitude of each triangle's surface normal projected along the positive Y axis.

Forward (+Z) Orientation. Shows scalar values proportional to the magnitude of each triangle's surface normal projected along the positive Z axis.

Vertex Normal Mapping. Toggles visibility of arrows oriented with polygon surface normals, the magnitudes of which depend upon the currenly selected scalar mapping.


Visualisations (toolbar)

Opacity. The leftmost spinbox widget on the toolbar (left of the World Axes button) specifies how opaque the selected model is. Maximum opacity is always 1.0 but the range is scaled based on context. In particular, if models are overlapping in the same viewer, the maximum opacity is limited so that both models are visible.

Surface Mapping Range. The next spinbox on the toolbar (right of the World Axes button) specifies the absolute range of the displayed values for the currently selected surface (scalar) visualisation. This widget is greyed-out if a surface visualisation is not currently visible on the selected model.

Number of Surface Mapping Colour Bands. The rightmost spinbox on the toolbar (immediately to the right of the Surface Mapping Range spinbox) specifies the number of colour bands shown for the selected surface mapping (i.e., the coarseness of the scalar bins) over the current range.


Camera

View Front Profile. Set the camera to view the subject's front facial profile, or the front side of the model if no landmarks are placed.

View Right Profile. Set the camera to view the subject's right facial profile, or the right side of the model if no landmarks are placed.

View Right Quarter. Set the camera to view the subject's right front quarter facial profile, or the right front quarter of the model if no landmarks are placed.

View Left Quarter. Set the camera to view the subject's left front quarter facial profile, or the left front quarter of the model if no landmarks are placed.

View Left Profile. Set the camera to view the subject's left facial profile, or the left side of the model if no landmarks are placed.

View From Above. Set the camera to view the subject from above.

View Top Quarter. Set the camera to view the subject from above and front.

View From Below. Set the camera to view the subject from below.

View Bottom Quarter. Set the camera to view the subject from below and front.

Orthographic Projection. Toggle between orthographic and perspective (default) projection. When the measurements are being visualised, orthographic projection is automatically toggled on (unless this behaviour is changed in the preferences).

Synchronise Cameras. If toggled on, the cameras for each viewer will be synchronised so that camera movement in one viewer is replicated across the others.

Marquee Mode. Toggles automatic rotating and panning of the cameras in each viewer about their foci. Useful for presentations. Normal mouse controlled camera interaction is overridden while this mode is active. Toggle off to return to normal interactio or click one of the predefined camera views (e.g., front view with the SPACE key).


Transform

Align Model. Try to align the model in a centred and upright position. If landmarks are set, the face is aligned in Frankfort horizontal (standard) position.

Centre Model. Set the centre of the model coincident with the zero (origin) point of the world axes.

Rescale. Rescale the model in one or more of the three spatial directions.

Reflect Laterally. Reflect the model through its YZ plane, or if landmarks are set through the mid-sagittal plane.

Rotate 90 degrees about X. Rotate the model anti-clockwise around the X axis by a quarter turn. If landmarks are set, rotation is with respect to the landmark defined alignment.

Rotate 90 degrees about Y. Rotate the model anti-clockwise around the Y axis by a quarter turn. If landmarks are set, rotation is with respect to the landmark defined alignment.

Rotate 90 degrees about Z. Rotate the model anti-clockwise around the Z axis by a quarter turn. If landmarks are set, rotation is with respect to the landmark defined alignment.

Manually Reposition. Enter model movement mode where mouse interactions that normally move the camera with respect to the model now move the model with respect to the camera. Clicking off the model reverts back to normal camera interaction mode.


Geometry

Invert Normals. Swap vertex ordering defined for the model's polygons which has the effect of inverting all surface normals. Use this where 3D capture systems do not agree on clockwise versus anti-clockwise vertex ordering of triangles in the underlying mesh.

Fill Holes. Finds and fills holes in the surface of the model. Holes are filled naively as a plane of triangles. This tool is meant to help fill small holes (of a few triangles) - not to reconstruct whole regions of the face.

Smooth Surface. Smooth surface geometry in areas of sharp curvature. The threshold for acceptible curvature should be set high enough to prevent important features from being lost. The smoothing threshold can be set in Cliniface's preferences. Mesh (vertex) connectivity is unaffected.

Extract Facial Region. Use after selecting a region with the radial selection tool to remove the parts of the model falling outside of the defined region. Alternatively, use without the radial selection tool to allow Cliniface to make its best guess as to the facial region. With landmarks set, and without using the radial selection tool, the facial region is extracted to be the minimum radially defined facial surface containing all of the landmarks.

Create Right Symmetric Face. Take the right side of subject's facial model, reflect a copy of it, and use this as the new left side of their face. Creates a perfectly symmetric face through the mid-sagittal plane if landmarks are set. If no landmarks are set, the YZ plane is used as the reflecting plane. The registered mask (if set) is not made symmetric and is invalid after this function. If you want to work with the symmetric version of the subject's face, you should discard the existing detection and redetect to ensure a better mask fit.

Create Left Symmetric Face. As above for the opposite half of the face.


Metrics

Detect Face/Landmarks. Detect the face and perform automatic landmark placement. This uses a non-rigid mapping of a template facial mask to the subject's face. The template mask (and its landmarks) can be set in Cliniface's preferences. If landmarks are already set, the user will be asked to select the ones to overwrite with new positions.

Discard Face Detection. Removes the facial landmarks and the mapped correspondence mask for all assessments.

Edit Landmarks. Display the landmark browser and unlock landmark positions to enable editing by left-clicking and dragging them to new positions. Landmark positins are relocked when this dialog is closed.

Show Landmarks simply toggles visibility of the landmarks.

Measurements Browser. Display the list of measurements taken from the face as well as where on the face they are measured. By default, projection mode is switched to orthographic when viewing measurements. See here for more information about selecting and making facial measurements.

HPO Terms Browser. Display the list of HPO terms that Cliniface is able to automatically identify through its measurements of the 3D facial surface. See here for more information about selecting and making facial measurements.

Show Calliper Measurements. Display any user defined calliper measurements for the current assessment. See here for more information about calliper measurements.

Discard ALL Calliper Measurements. Deletes all calliper (user defined) measurements from the current assessment for the active model.


Help

Help. The help system you're currently looking using.

What's This?. Some functions have context sensitive information - click this button then on a function to find out more about it.

Preferences. Display the preferences dialog to configure Cliniface's behaviour.

Plugins. Display the loaded plugins.

About. About Cliniface, how to cite it, and how to support our efforts.


Context (Right Click Menu)

Set Camera Focus to the location pointed at by the mouse upon right clicking. This can also be achieved by pressing F with the mouse pointer hovering over the new point to focus the camera upon.

Add Calliper Measurement between two points on the surface of the model. The calliper measurement is added as a user defined measurement to the current assessment. Click and drag the end-points to place the measurement. See here for more information about calliper measurements.

Rename Calliper Measurement. Calliper measurements are unnamed by default. Give them a name so they are easier for others to reference (and to retrieve from exported XML/JSON data).

Discard Calliper Measurement. Delete the calliper measurement whose handle was under the mouse pointer at the time of right clicking.

Restore Landmark Position. Restore the position of the landmark under the mouse pointer to the automatically detected position. Only available if the Landmarks Browser is open allowing landmarks to be repositioned.

Select Radial Area. Open the radial selection tool centred on the position under the mouse cursor. If already on, selecting this turns radial selection mode off.

Discard Manifold. Discard the geometric manifold connected to the region under the mouse pointer.

Discard OTHER Manifolds. Discard the geometric manifolds other than the one connected to the region under the mouse pointer.
