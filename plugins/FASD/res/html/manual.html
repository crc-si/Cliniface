The Fetal Alchohol Spectrum Disorder module for Cliniface is based upon the diagnostic assessment methodology using the FASD 4-Digit Code for 2D images of faces developed by Susan J. (Astley) Hemingway, Ph.D, Professor of Epidemiology/Pediatrics, and Director, WA State FAS Diagnostic & Prevention Network (FAS DPN). While based upon this methodology, its application in Cliniface is for the measurement and assessment of 3D facial models rather than of 2D facial images. Because this methodology is still being validated in the 3D domain, the results it generates should not be considered to have the same level of reliability as when the methodology is applied in its original 2D image domain. Neither Dr. Hemingway, nor any of her affiliates were consulted during the development of this plugin.

Using the FASD Module

The FASD module is loaded if it is visible as an entry under Cliniface's Metrics menu. The image below shows the FASD dialog that is shown when selected. The option is greyed out unless the selected model has facial landmarks present. Information in the dialog is segmented into three sections: Palpebral Fissure Lengths (PFL), Upper Lip Fullness, and University of Washington Facial Diagnostic Rankings. Each section is described below.

Palpebral Fissure Lengths (PFL)

The left, right, and mean PFL lengths are shown according to the current model measurements of the distance between the exocanthian (outer eye) and endocanthian (inner eye) landmarks.

The Z-scores to the right of the measurement values are calculated according to the currently set statistics for the PFL. Hovering the mouse over any of the z-score values shows the current reference for the source of the growth curve statistics being used to calculate these values. To change the source of the PFL growth curve statistics, open the Measurements Browser from the Metrics menu, select the PFL metric from the table, uncheck the Automatch checkbox if checked, and select the desired source from the dropdown list.

All of the statistics in Cliniface provide the measurement as a point-to-point distance between the canthi of each eye. By checking In-plane in the Measurements Browser, the PFL distances will be recalculated as the projected distance into a plane parallel with the coronal plane. Due to the nature of such projections, the calculated values will be smaller by a factor of cos θ where θ is the angle of incidence between the PFL line and the coronal plane. The 2D values calculated in this way are comparable to the PFL distances measured from 2D images of faces (not accounting for the additional error factor due to perspective projection).

A short mean palpebral fissure length results in a higher FASD diagnostic ranking.

Upper Lip Fullness

The fullness of the upper lip (or circularity) is calculated as the square of the upper lip's perimeter by its area. This metric was originally designed to be used in the context of 2D images by having the user trace the perimeter of the upper vermilion in a photograph. Larger values denote a thinner upper lip.

In Cliniface, the perimeter of the upper lip is estimated using the landmarks around the vermilion. These landmarks are Stomion Superius (SS), Left/Right Mid-labial Fissure Superius (MF), Left/Right Cheilion (C), Left/Right Mid-vermilion Superius (MV), Left/Right Crista Philtri (CP), and Labrale Superius (LS). The area is calculated as the sum of the areas of the eight "inner" triangles as shown in the diagram below.

While the automatic demarcation of the perimeter using these landmarks is not as precise as tracing the path exactly around the upper vermilion, the major components of the upper lip contributing to its fullness score are captured by this method without having to manually indicate where to exactly draw the upper lip perimeter. In addition, a 2D image projection of the upper lip may (depending on the orientation of the mouth) inaccurately show the degree of curvature of the upper or lower edges of the upper lip as being more or less horizontally curved than in reality. Projecting this metric into the frontal plane (simulating measurement from a 2D image) results in the fullness metric being calculated differently than in 3D.

The Ethnicity Guide should be manually set to one of the two available guides: Caucasian, or African American so that the correct ranking can be scored by thresholding on the circularity value. The thresholds used are the same as those for the 2D image guides about which more information can be found on page 30 of the Diagnostic Guide For Fetal Alcohol Spectrum Disorders: The 4-Digit Diagnostic Guide.

University of Washington Facial Diagnostic Rankings

The University of Washington's FASD ABC 1-5 ranking system associates a numeric rank from 1 to 5 and a scoring letter to each of the three sentinel features. The philtrum curvature rank must be manually entered with reference to page 30 of the Diagnostic Guide For Fetal Alcohol Spectrum Disorders: The 4-Digit Diagnostic Guide ensuring that the same ethncity guide is used as set in the Upper Lip Fullness section of the dialog. The image guides are not included in this software due to copyright restrictions.

Generating the FASD Report

The FASD report can be generated either directly from the FASD dialog by clicking the report generation button at the bottom left, or via Cliniface's report selection dialog accessible from the File menu or using the toolbar button. The report summarises the content of the FASD dialog box described here, generates and embeds the PFL growth curve statistics used, and also embeds a 3D model of the subject's face together with demographic and assessment information.
